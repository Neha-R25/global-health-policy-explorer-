Interactive R Shiny dashboard exploring mortality trends alongside public health policies and interventions across countries.

# Overview
This dashboard provides an exploratory visualizayion of mortality trends alongside major public health policies and intervention across countries.
The tool integrates country-level mortality data with policy timelines to support a structured view of how public health strategies align 
with disease trends.
The project was developed to demonstrate how policy timelines and epidemiological indicators can be visualized together in an interactive 
analytical interface. 

# Key features
Interactive country and cause selection
Visulaization of mortality trends over time
Policy and intervention timeline markers integrated into mortality plots
Latest mortality rate indicator for selected country and cause
Interactive table of policy and intevention records
Designed as an exploratory anlytical tool for public health research

# Research context 
Understanding how mortality trends evolve alongside national health policies is an important component of public health evaluation and policy analysis. This dashboard was developed as an exploratory tool to visually examine temporal relationships between:

national public health policies
health promotion interventions
disease-specific mortality trends

The project demonstrates how data visualization tools can support exploratory public health analysis and policy awareness.

# Tools and technologies 
The dashboard was built using:
R
Shiny
tidyverse
ggplot2
DT
These tools were used for data cleaning, transformation, visualization, and dashboard development.

# Data Sources 
Mortality datasets were prepared using publicly available data derived from:

IHME – Global Burden of Disease (GBD) datasets

Policy and intervention timelines were curated from publicly available sources including:

national public health reports
government health ministry publications
public health policy documentation

# Project purpose 
This project was developed as part of an independent effort to explore how interactive data visualization tools can support public health policy analysis.

It demonstrates practical experience with:

Data cleaning and preparation in R
• Exploratory data analysis
• Public health policy research
• Data visualization using ggplot2
• Interactive dashboard development using Shiny
• Integration of policy timelines with epidemiological indicators
